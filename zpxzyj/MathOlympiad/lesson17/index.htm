∑ ｒ进制数转十进制数

例⒈　把七进制数３２６１０转化为十进制数。
∞

解答

表示３２６１０是七进制数用３２６１０７。几进制就是逢几进一。七进制数，个位数７次后进第２位，第２位的１表示７。第２位数７次后进第３位，第３位的１表示７×７＝４９。第３位数７次后进第４位，第４位的１表示４９×７＝３４３……以此类推。

(32610)7 = 74×3 + 73×2 + 72×6 + 7×1 + 0 = 8190

∫ 十进制数转ｒ进制数

例⒉　把６７１０２转化为六进制数。
∞

解答

先把６７１０２放到六进制数的个位，可是六进制数的个位只能是０到５，逢６就要进１。那么６７１０２要进多少次１呢？

67102÷6=11183......4
∞

１１１８３次。余下的４就是个位。１１１８３再放到第二位，每有一个６就得向第三位进１。

11183÷6=1863......5
∞

第三位现在是１８６３。余数５是第二位。以此类推，继续“延展”到更高位。

1863÷6=310......3

310÷6=51......4

51÷6=8......3

8÷6=1......2

1÷6=0......1

∞

称为除ｒ取余法。把所得的余数序列倒过来写，就是答案：１２３４３５４。

∆ 同余的性质

例⒋　－２除以３的余数是多少？
∀

概念

当xmodz=ymodz，我们说x≡y(modz)。读作“ｘ与ｙ模ｚ同余”。例如４３和５８除以３的余数都是１，就说43≡58≡1(mod3)。两个同余等式可以相加减或相乘。（我可没说过可以做除法。）

π

解答

我们知道

{-2≡-2(mod3)①0≡3(mod3)②

①＋②得

-2≡1(mod3)

也就是说，－２除以３的余数和１除以３的余数一样。答案就是１。

!

启发

余数与除数同号。被除数可增减除数任意多次，余数不变。

π

解答

一个周期６颗珠子，模数就是６。

30≡6(mod6)

第３０个珠子的颜色与第６颗相同，因此是黑色。
